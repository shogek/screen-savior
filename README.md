# screen-savior

## How to experience the magic
1. `git clone https://github.com/shogek/screen-savior.git`
1. `cd screen-savior`
1. `npm install`
1. `npm run start` (to generate the JavaScript file)
3. `index.html`

## How to configure
1. Open `/scripts/_settings.js`
1. Make wanted changes
1. Reload website

## How to develop
1. `npm install`
1. `npm run start`
